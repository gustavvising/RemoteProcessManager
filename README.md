# RemoteProcessManager

A Windows remote process management tool with a client-server architecture and a web-based graphical dashboard for monitoring and handling system processes.

⚠️ Under development

## Roadmap
- Implement WebSocket communication in C++ client  
- Design and implement Go backend server  
- Define client–server message protocol  
- Build React dashboard UI  
- Add authentication layer for remote access  
- Improve logging and error handling across components
